# 💬 Sistema de Emoji com PHP + SQLite

Projeto web desenvolvido em PHP utilizando SQLite como banco de dados.
O sistema permite criar mensagens e reagir com emojis, semelhante a redes sociais.

---

## 🚀 Funcionalidades

* ✍️ Criar mensagens
* 👍 Reagir com emojis (👍 ❤️ 😂)
* 📊 Contagem de reações por mensagem
* 💾 Banco de dados SQLite automático
* 🔒 Código seguro com Prepared Statements

---

## 🛠 Tecnologias utilizadas

* PHP
* SQLite
* HTML + CSS

---

## 📁 Estrutura do projeto

```
emoji-system/
│
├── index.php
├── conexao.php
├── postar.php
├── reagir.php
├── style.css
└── banco.db
```

---

## ▶️ Como rodar o projeto

1. Instale o XAMPP
2. Coloque a pasta dentro de:

```
htdocs/
```

3. Inicie o Apache no XAMPP

4. Acesse no navegador:

```
http://localhost/emoji-system
```
